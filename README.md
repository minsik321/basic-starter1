## 개선점

- 서로 다른 객체의 내용에 대해 동일시 하려는 경우에는 같은 type의 객체로 선언해야 함.
- 빼빼로의 길이와, 토핑이 바뀔 수도 있다고 생각하고, 모든 경우를 만들어 두는 게 아닌 커멘트 라인에서 길이, 토핑, 바디 를 모두 입력 받는 구조로 바꿈.
